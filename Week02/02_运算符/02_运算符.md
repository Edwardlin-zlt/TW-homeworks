## 自己搜索查询 num++（后面自加）和++num（前面自加）的区别，并进行总结。

`++` 自增运算符

如果放在操作数前面，则先加后用; 如果放在操作数后面，则先用再加。
比如：

```
var x=3;

console.log(++x); // 返回4

console.log(x) // 返回4

// ---分割线---

var x=3;

console.log(x++); // 返回3

console.log(x); // 返回4
```

## 自己搜索查询如何利用 Math 方法对得到的结果进行保留两位小数，四舍五入，向上取整，向下取整等操作，并进行总结。

`Math` 是一个内置对象， 它具有数学常数和函数的属性和方法。不是一个函数对象。

`Math` 适用于 `Number` 类型。它不支持 `BigInt`。

`Math` 的所有属性与方法都是静态的。

1. 保留两位小数 `Math.round(value*100) / 100`
   `Math.round()`将根据四舍五入取整，如果需要保留两位小数，先将数字 ×100,在`round`取整后/100 得到最后的结果。(如果没有数字刚好为整数，没有小数部分则不会显示.00,可以通过转换为字符串进行格式化输出)

   拓展： 四舍五入保留 n 位小数

   ```
   function roundFun(value, n) {
       return Math.round(value*Math.pow(10, n)) / Math.pow(10, n);
   }
   ```

   拓展: 保留两位小数格式化输出

   ```
   function roundFun(value, n) {
       var tmp_num = Math.round(value*Math.pow(10, n)) / Math.pow(10, n);
       return tmp_num.toFixed(2);
   }
   ```

2. 向下取整 `Math.floor(value)`
   保留小数的方法同上

3. 向上取整 `Math.ceil(value)`
   保留小数的方法同上

## 分别指出以下四处变量 a 对应的值？并说明为何等于该值。

JavaScript 的`number`数据类型是`双精度IEEE 754 64位浮点类型`.
按位与`&`，按位或`|`，按位非 都是根据二进制位数来进行计算运算的。

```

var a = 42 & 24;
// 此时 a 的值为: 8
// 42  -> 101010 转换二进制
// 24  -> 011000 转换二进制
// out -> 001000 -> parseInt('001000', 2) -> 8

a = a && 34;
// 此时 a 的值为: 34
// 逻辑运算与`&&`，如果第一个计算数为`false`则返回第一个计算数，否则返回第二个计算数。

a = a | 54;
// 此时 a 的值为： 54
// 按位与：同一二进制为有一则为1,全0则为0
// a   -> 100000
// 54  -> 110110
// out -> 110110 -> parseInt('110110', 2) -> 54

a = a || 68;
// 此时 a 的值为： 54
// 逻辑运算或`||`, 如果第一个计算数为`true`则返回第一个计算数，否则返回第二个计算数。

```
